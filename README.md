# Day 1 - User CRUD API

A simple REST API built using ASP.NET Core for Day 1 of my 30-Day API Challenge.

## Endpoints
GET /api/users
GET /api/users/{id}
POST /api/users
PUT /api/users/{id}
DELETE /api/users/{id}

## Tech Stack
- ASP.NET Core Web API (.NET 8/9)
- In-Memory List
- Services + Controllers Pattern
- Swagger UI
